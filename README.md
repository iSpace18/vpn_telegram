# VPN Telegram Bot

Telegram-бот для автоматической продажи VPN-доступа с интеграцией панели Marzban (или 3x-ui/Remnawave) и оплатой через Telegram Stars.

## Возможности

- Просмотр тарифов и покупка подписки через Telegram Stars.
- Автоматическое создание VPN-пользователя на панели после оплаты.
- Пробный период (опционально).
- Реферальная система с уникальными ссылками и бонусами.
- Административная панель внутри бота:
  - Статистика.
  - Управление серверами.
  - Ручное управление ключами.
  - Рассылка сообщений.
- FAQ и поддержка.

## Требования

- Python 3.10+
- Docker и Docker Compose (рекомендуется)
- Панель управления VPN (например, [Marzban](https://github.com/Gozargah/Marzban))
- Telegram Bot Token от [@BotFather](https://t.me/BotFather)
